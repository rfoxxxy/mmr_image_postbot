# mmr_image_postbot

# Запуск
1. Установите необходимые библиотеки
`pip install -r requirements.txt`
2. Создайте файл config.py и поместите туда токен бота.
_Пример содержания файла:_
```
TOKEN = "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
```
3. Запустите бота
`python3 bot.py`
